# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Added `CONTRIBUTING.md` with contribution guidelines
- Added `CHANGELOG.md` to track project changes

## [0.0.5] - 2024-01-XX

### Fixed
- Fixed typo: `lenght` → `length` throughout codebase (10 occurrences)

### Added
- Added comprehensive documentation to all public widget structs
- Added module-level documentation to prelude
- Added documentation to key enums (Signal, Align, Cursor)

## [0.0.4] - 2023-12-XX

### Added
- Initial widget implementations
- Container widgets: Box, Naviframe, Frame, Panes
- Output widgets: Label, Separator, ProgressBar
- Input widgets: Button, Check, Entry
- Range widgets: Slider, Spinner
- Selector widgets: Radio, List, SegmentControl, Menu
